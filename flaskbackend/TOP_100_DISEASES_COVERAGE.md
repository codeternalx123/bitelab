## System Capabilities

### ✅ **Multi-Disease Optimization**
- Handles unlimited diseases simultaneously
- Family-level meal planning (up to 10+ members)
- Automatic conflict resolution for nutritional guidelines
- Performance: <5ms for 20+ diseases

### ✅ **Evidence-Based Guidelines**
- ICD-10 medical coding for all conditions
- Nutritional guidelines from medical authorities:
  - WHO, FDA, ADA, AHA, NKF
  - Academy of Nutrition and Dietetics
  - Disease-specific medical associations
- Peer-reviewed clinical research

### ✅ **Comprehensive Nutritional Profiles**
Each disease includes:
- **Nutritional Guidelines**: Macro/micronutrient targets with priorities
- **Food Restrictions**: Severity-based with alternatives
- **Recommended Foods**: Evidence-based food lists
- **Meal Timing**: Critical for diabetes, GI, kidney diseases
- **Portion Control**: Essential for CKD, diabetes, cancer
- **Hydration Requirements**: Disease-specific fluid needs

### ✅ **Smart Conflict Resolution**
When family members have conflicting dietary needs:
- Priority-based guideline merging (critical > high > medium > low)
- Most restrictive food rules take precedence
- Balanced nutritional targets across all conditions
- Alternative food suggestions for restrictions

## API Endpoints

**Disease Database API:** `http://127.0.0.1:5001`

```
GET  /v1/diseases/list              # All 175 diseases
GET  /v1/diseases/categories        # 12 categories
GET  /v1/diseases/search?q=diabetes # Search by name/ICD-10
GET  /v1/diseases/<id>/details      # Complete disease profile
```

**Main Meal Planning API:** `http://127.0.0.1:5000`

```
POST /v1/meal-planner/generate      # AI meal planning with disease optimization
```

**Example Request:**
```json
{
  "family_members": [
    {
      "name": "John",
      "age": 45,
      "diseases": ["type2_diabetes", "hypertension"],
      "dietary_preferences": "non-vegetarian"
    },
    {
      "name": "Mary",
      "age": 42,
      "diseases": ["celiac", "osteoporosis"],
      "dietary_preferences": "vegetarian"
    }
  ],
  "cuisine": "Mediterranean",
  "duration_days": 7,
  "family_size": 4,
  "goals": ["weight_loss", "heart_health", "bone_health"]
}
```
